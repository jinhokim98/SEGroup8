#### Actor descriptions (추가 상품 구매 ~ 상품 판매/구매 통계)

| Actor                                | Description                                                  |
| :----------------------------------  | ------------------------------------------------------------ |
| 판매자                                 | 판매 완료된 자신의 상품을 조회할 수 있다.                             |
| 구매자                                 | 상품정보를 조회할 수 있고, 상품을 구매할 수 있다.                       |
| 환불 event                            | 상품이 환불되는 경우 적립되었던 포인트를 반환한다.                        |
